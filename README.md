# 🏭 Nassau Candy Factory Reallocation & Shipping Optimization System

An AI-powered decision intelligence dashboard developed for **Nassau Candy Distributor** to improve shipping efficiency, reduce lead times, and optimize factory-product allocation using **Machine Learning, Data Analytics, and Streamlit**.

---

## 📌 Project Overview

Nassau Candy Distributor uses static legacy factory-product assignment methods, which can lead to:

- High shipping lead times
- Increased logistics cost
- Inefficient regional fulfillment
- Reduced profitability

This project solves the problem by building a smart recommendation system that predicts shipping performance and suggests better factory allocations.

---

## 🎯 Objectives

- Predict shipping lead time using historical order data
- Analyze logistics bottlenecks by region and ship mode
- Recommend better factory-product assignments
- Improve operational efficiency without hurting profit
- Provide an interactive dashboard for decision-making
